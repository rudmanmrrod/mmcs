Introducción

Para el año 2015, desde la Fundación Centro Nacional de Desarrollo e Investigación en Tecnologías Libres, se desarrolló una primera versión de un software destinado al manejo de las matrices de contabilidad social, entendiendo que actualmente se maneja por medio de macros en hojas de calculo, dificultando la visualización de los resutados.

El Sistema para el manejo de las Matrices de Contabilidad Social (SMCS), fue diseñado con el objetivo de suministrar a los analistas, investigadores, un software libre que permita construir los modelos de multiplicadores clásico, de precios homogeneos, precios no homogeneos y evaluar escenarios. Además de facilitar la visualización de la información.


SMCS, fue desarrollado bajo el lenguaje de programación C++, utilizando el framework Qt. El código fuente SMCS está distribuido bajo la licencia GPL v 2.0 de la Free Software Foundation. Puede acceder al gestor del proyecto por medio de miv.cenditel.gob.ve/mmcs

La arquitectura del sistema cuenta con tres capas:

  • Carga de archivos y vizualización de resultados: se encarga de gestionar todas las solicitudes que realiza usuario a la aplicación y de mostrar los resultados de estas operaciones.
  • Organización y almacenamiento de datos (matrices): se encarga de transformar los datos cargados a través de la primera capa en estructuras de datos matriciales que serán trabajadas por la siguiente capa y de entregar los datos de resultados a la primera capa para ser visualizados.
  • Operaciones matriciales: se encarga de realizar las distintas operaciones disponibles para el usuario con los datos suministrados, estas en su mayoría son operaciones matriciales que se realizan con la biblioteca GSL - GNU Scientific Library

    • Además, cuenta con una capa trasversal que contiene la configuración de ciertos parámetros generales para la aplicación que pueden ser accedidos desde cualquiera de las capas anteriores, y algunos de los cuales con definidos por el usuario.

    Menú Principal

    El software cuenta con cuatro elementos en el menú:

  • Archivo: Al ubicar el cursor del ratón sobre la palabra Archivo, se le desplegará un submenú con cuatro elementos. El primero de ellos, Nuevo Proyecto le abrirá otra vez el sistema. Cargar Matriz, le permitirá cargar la matriz que objeto de estudio (archivo .csv), si desea exportar una matriz utilice Exportar Matriz y por último, presione Salir para cerrar el sistema.
  • Herramientas: En esta parte del menú, encontrará todas las funcionalidades del sistema: estimación de coeficientes técnicos, multiplicadores de leontief, clasificador de cuentas, y evaluación de escenarios por medios de los modelos clásico, no clásico, precios homogéneos clásico y no clásico (incidencia 100% y al i%) y precios no homogéneos (incidencia 100% y al i%)
  • Visualización: Este elemento del menú le permitirá seleccionar una submatriz o una parte de esta, para luego mostrarsela en una pestaña.
  • Ayuda: Por medio de este elemento, podrá acceder al manual de usuario y obtener información adicional del sistema.
